Final Case Study: Image Collage with Filters (Tkinter)

📌 Overview

This project is a final case study developed using concepts learned during classroom sessions.
It focuses on creating an image collage application with filter effects using Python and the Tkinter GUI framework.

The application allows users to combine multiple images into a single collage and apply visual filters to enhance the appearance.

---

🎯 Objectives

- To design a graphical user interface using Tkinter
- To implement image collage creation
- To apply image processing techniques such as filters
- To integrate user interaction with image transformations

---

⚙️ Features

- Graphical User Interface (GUI) using Tkinter
- Selection of multiple images
- Creation of image collage
- Application of filters such as:
  - Grayscale
  - Brightness adjustment
  - Contrast enhancement
- Real-time visual output
- Interactive buttons for user control

---

🛠️ Technologies Used

- Python
- Tkinter (GUI Framework)
- Pillow (PIL) / OpenCV (if used)

---

🧠 Concepts Applied

- GUI development using Tkinter widgets
- Event-driven programming
- Image processing techniques
- Pixel manipulation and transformations
- File handling
